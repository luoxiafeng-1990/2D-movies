调色板查找型 (Palette Lookup)
原理：像素值本身不是颜色，而是调色板 RAM 的索引号。硬件用索引查表得到 R8G8B8。
BPPMODE枚举值格式名称每像素颜色数内存布局转换方式
0000001BPP Palette1 bit2每 bit 一个像素索引索引 → 查 Palette RAM → R8G8B8
0000112BPP Palette2 bit4每 2bit 一个像素索引索引 → 查 Palette RAM → R8G8B8
0001024BPP Palette4 bit16每半字节一个像素索引索引 → 查 Palette RAM → R8G8B8
0001138BPP Palette8 bit256每字节一个像素索引索引 → 查 Palette RAM → R8G8B8
低位数 RGB 扩展型 (Bit Expansion)
原理：R/G/B 各分量位数不足 8 位，硬件将低位数扩展到 8 位（MSB 复制填充 LSB）。带 Alpha 的格式，Alpha 用于 blending。

YUV 色彩空间转换型 (Color Space Conversion)
原理：像素数据是 YUV 格式，硬件通过内置的 YUV→RGB 矩阵运算实时转换为 R8G8B8。
BPPMODE枚举值格式名称内存布局转换方式
1000117YUV420SP (NV12)Y平面: [YYYY...] + UV交错平面: [UVUV...]硬件矩阵运算 → R8G8B8
YUV→RGB 转换公式（硬件矩阵实现）
R = Y + 1.402 × (V - 128)
G = Y - 0.344 × (U - 128) - 0.714 × (V - 128)
B = Y + 1.772 × (U - 128)
