为什么训练"死"了？

z = w × 200 太大 → sigmoid 完全饱和
σ(500) = 1.00000000（完全等于 1）
σ'(500) = 1×(1-1) = 0
dL/dw = ... × 0 × ... = 0
梯度为 0 → 权重无法更新 → 永远卡住

为什么训练"活"着？

z = w × 0.784 很小 → sigmoid 在活跃区域
σ(0.08) = 0.52（中间地带）
σ'(0.08) = 0.52×0.48 = 0.25
dL/dw = ... × 0.25 × ... = 合理值
梯度健康 → 权重稳步更新 → 持续学习
